# 痛風メモ（Gout Memo）

痛風の自己管理用 PWA。iPhone / Android のブラウザからホーム画面に追加して使える、軽量・オフライン・local-first のアプリです。

## なぜこれ?

既存の痛風アプリは「尿酸値トラッキング」「食事解析」「AI診断」などの機能を並べがちですが、**毎日続けるのが難しい**のが最大の問題です。このアプリは逆に振り切っています:

- **1日の記録が 3タップ〜15秒** で終わる設計
- **痛みと部位**に最も強い（人体図/足の図からタップ選択、左右別）
- **発作前24h / 48h の行動**を自動で振り返れる
- 医師への**受診レポート（30/90日、発作サマリー、CSV、印刷）**が1クリック
- **完全 local-first**。ログイン不要、サーバー送信ゼロ、オフライン動作

## 主な機能

- ワンタップ記録: 「飲まなかった」/ 水分 +250ml / 痛み / 薬
- 痛みの強さ (NRS 0〜10) + 部位タップ (足/全身, 左右両側) + 腫れ/熱感/歩行支障
- 発作の自動開始・終了、過去の発作タイムライン
- 飲酒量（種類別の簡易選択）、水分、高リスク食品（内臓・魚卵・赤身肉・甘い飲料 など）、体調（睡眠不足・ストレス・脱水・運動しすぎ）
- 服薬管理（毎日服用/頓服、時刻リマインダー、メモ）
- タイムライン可視化 + 控えめな「傾向」表示（「関係があるかもしれません」）
- 受診レポート（30日 / 90日 / 発作サマリー、CSV・JSONエクスポート、印刷/PDF）
- 日本語 UI、ダークモード、文字サイズ変更、大きめボタン
- PWA: ホーム画面追加、オフライン動作、起動が速い

## 使い方

静的ファイルのみなので、HTTP サーバーで配信すれば動きます。

```bash
# 例: Python で手元確認
python3 -m http.server 8000
# → http://localhost:8000/ を iPhone/Android で開き「ホーム画面に追加」
```

> Service Worker を使うため、`file://` では動作しません。https か localhost で配信してください。

## 重要な注意

このアプリは **医療行為の代行や診断を行いません**。服薬の可否・用量は必ず医師の指示に従ってください。本アプリが表示する「傾向」は統計的推論ではなく、あなたの記録内の単純な相関表示です。

## データ

- 端末内 `localStorage` にのみ保存されます（外部送信なし）
- JSON でバックアップ / 復元可能
- CSV でエクスポート可能
- 設定画面からデータ全消去可能
